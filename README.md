# Design Specialist

**AI駆動で独自性のあるデザインを生成する実験プロジェクト**

このプロジェクトは、Claude（AI）が主導で、ジェネリックな「AIっぽい」デザインを超えた、**明確な美的方向性**を持つプロダクショングレードのUIを生成できることを実証します。

## プロジェクトのコンセプト

### AI駆動デザインの課題と挑戦

多くのAI生成デザインは、以下のような特徴に収束しがちです：

- 使い古されたフォント（Inter、Roboto、Arial）
- 陳腐なカラースキーム（白地に紫グラデーション）
- 予測可能なレイアウトパターン
- 個性に欠けるクッキーカッターデザイン

**このプロジェクトの挑戦**: AIが**意図的**に極端な美的方向性を選び、記憶に残る独自のデザインを生成できるか？

### デザイン生成プロセス

Claude（AI）は以下のプロセスでデザインを生成します：

1. **Purpose（目的）** - ユーザー、ブランド、コンテキストを理解
2. **Tone（トーン）** - 極端な美的方向性を選択（ミニマリズム、最大主義、レトロフューチャー、オーガニックなど）
3. **Constraints（制約）** - 技術要件、アクセシビリティ、パフォーマンスを考慮
4. **Differentiation（差別化）** - ユーザーが記憶する1つの要素を定義

詳細なガイドラインは [CLAUDE.md](./CLAUDE.md) を参照。

## 技術スタック

## 生成されたデザインサンプル

### デザインパターン集 (`/patterns`)

AIが生成した様々な美的方向性の実装例：

- **Retro-Futuristic** - レトロフューチャーな世界観
- **Brutalist** - ブルータリスト・生々しい表現
- **Editorial** - エディトリアル・雑誌風レイアウト
- **Organic** - オーガニック・自然的なデザイン
- **Generic** - 避けるべきジェネリックなデザイン（反面教師）

### Organic Site (`/organic-site`)

Organicデザイン思想に基づいた完全なマルチページサイト：

- **トップページ** - ブランドストーリーとビジュアル体験
- **About** - チームと価値観の紹介
- **Services** - サービス内容と料金プラン（スマホ最適化済み）
- **Contact** - お問い合わせフォーム

**AI生成デザインの特徴**:
- 自然な曲線とオーガニックな形状
- アースカラーを基調とした優しい配色
- 流れるようなスクロールアニメーション
- IntersectionObserverを活用したインタラクション
- 完全なスマートフォンレスポンシブ対応

## デザイン原則

AIが従う3つの原則：

1. **多様性の重視** - 同じデザインは作らない。プロジェクトごとに異なるフォント、カラー、レイアウトを選択
2. **収束を避ける** - トレンドフォント（Space Grotesk、Instrument Sansなど）を避け、常に新鮮な選択を
3. **ビジョンの実行** - 実装の複雑さを美的ビジョンに一致させる（マキシマリスト ↔ ミニマリスト）
